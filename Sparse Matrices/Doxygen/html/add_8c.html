the first thing we need to do is to check that the 2 matrices have the same size, as otherwise we cannot add them

now we need to calculate the resulting matrix first we need to know its size, as it is the sum of the sizes of the other 2 matrices - number of common elements

first we check if they are on the same row

now we check if they are on the same column

we know that we have 2 elements located at the same position now we have to check if their sum if 0 in order to decrease the number of elements by 2(as we need to exclude this element from the new matrix), or not, to decrease it by 1

now we do the addition itself

first we check if the elements of the original matrices are on the same row and if not we just add the current element to the result matrix

now we check if they are on the same column

as they are located as the same position, we add to the resulting matrix their sum

if it is 0 we do not add anything
